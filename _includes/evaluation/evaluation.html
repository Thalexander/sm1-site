Compatibility

By using Microsoft Bot framework rather than directly using the apis of the various instant messaging services, we can support a large array of different messaging platforms. As a result of our research and choosing this framework, we have great compatibility.


Maintainability

The bot is highly maintainable as it uses a single api that is maintained by Microsoft and has a single code base to traget multiple platforms as opposed to different bots for each platform if we had used their individual apis.


Evaluation of Testing

We did not follow test driven development, however we did use testing in some cases to ensure the system operated as expected. Whilst our code is quite minimal so we can be quite confident that it will operate as expected, we hsould have followed a test-driven development methodology, such that we can be very confident in its behaviour.
